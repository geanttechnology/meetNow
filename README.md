# MeetNow
MeetNow is a full-stack web application inspired by Meetup. Built using:
- Ruby on Rails
- ES6
- React + Redux
- PostgreSQL database

If you're passionate about something, why wait?
Find exciting people who share the same energy as you and MeetNow!
